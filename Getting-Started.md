# Getting Started

## Access

- Get your GitHub account added to the org
- Request cloud provider access
- Set up VPN if required

## Tools

| Tool | Purpose |
|------|---------|
| `gh` | GitHub CLI |
| `terraform` | Infrastructure as code |
| `kubectl` | Kubernetes CLI |
| `helm` | Kubernetes package manager |
| `k9s` | Kubernetes terminal UI |
| `docker` | Containers |

Install with Homebrew: `brew install <tool-name>`

## Repositories

| Repo | Description |
|------|-------------|
| `app-frontend` | Frontend web application |
| `api-core` | Core backend API |
| `api-support` | Support API |
| `infra-modules` | Reusable Terraform modules |
| `infra-root` | Root infrastructure |
| `cicd` | Shared CI/CD templates |

## Steps

1. Clone the repos you need
2. Run `gh auth login`
3. Read the `README.md` in each repo
4. Join your team's Slack/Teams channel
